h
================================================================================

Display help for any command


Requirements
--------------------------------------------------------------------------------

Use `/bin/sh`. Tested on OSX and Ubuntu.


Installation
--------------------------------------------------------------------------------

Download to your executable path.

    $ cd $HOME/bin
    $ curl -LO https://raw.github.com/Tomohiro/h/master/h
    $ chmod +x ./h


Usage
--------------------------------------------------------------------------------

    $ h rbenv         # rbenv help
    $ h gem           # gem -h
    $ h irssi         # irssi --help
    $ h heroku addons # heroku addons --help
