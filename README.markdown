h
================================================================================

Display help for any command

Requiremetns
--------------------------------------------------------------------------------

Use `/bin/sh`. Tested on OSX and Ubuntu.


Installation
--------------------------------------------------------------------------------

Download to your executable path.

    $ curl -LO https://raw.github.com/Tomohiro/h/master/h $HOME/bin/


Usage
--------------------------------------------------------------------------------

    $ h rbenv # same: rbenv help
    $ h gem   # same: gem -h
    $ h irssi # same: irssi --help
